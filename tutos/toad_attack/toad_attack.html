Babylon.js is a 3D game engine based on WebGL and javascript, created by David Catuhe and David Rousset.
This engine is relatively new and less popular than Three.js, but many think it is easier to understand and easier to manipulate. Moreover, it is completely open source (with a very reactive community) and game oriented, with several features built specifically for games, likegamepad camera,hardware accelerated instances, and so on.
The complete list of feature is described on theGithub page of the project.
With this framework, you can build gameslike this one, based on the popular licenceAssasin's creed.
